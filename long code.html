Single-file, responsive, accessible, and useful — perfect for Notepad

This template gives you a clean starting point with components, utilities, and a little vanilla JavaScript. No external files or libraries required.

Tip: Save this as index.html from Notepad and open it in your browser.

Quick info

  • Theme: Light/Dark with localStorage
  • Nav: Responsive with a menu toggle
  • Components: Tabs, Accordion, Modal
  • Extras: Todo list & note saver

Features

Everything you need to kickstart a project directly from Notepad.

Responsive grid

Use the .grid, .grid-2, and .grid-3 utilities for quick layouts at different breakpoints.

Accessible components

Tabs, accordion, and modal all include ARIA attributes and keyboard-friendly behavior.

Utility classes

Grab handy classes like .card, .btn, .muted, .lead, .surface for consistent styling.

Tabs

Tabs show and hide content sections without reloading the page. They also support keyboard navigation.

Use ARIA roles and manage aria-selected for the active tab. Keep focus behavior intuitive.
